#  Kenyan Language Identifier

This app can detect the language of any short text written in English, Kiswahili, Sheng, Kikuyu, Luo, Kalenjin, or a mix of these languages.

---

## How to Run the App on Your Computer

1. Make sure Python is installed on your computer
2. Open your terminal and navigate to the project folder
3. Install the required packages by running:
   ```
   pip install streamlit scikit-learn pandas numpy
   ```
4. Start the app by running:
   ```
   streamlit run app.py
   ```
5. A browser window will open automatically with the app

---

## How to Put the App Online (Free)

1. Go to [share.streamlit.io](https://share.streamlit.io) and sign in with your GitHub account
2. Click **"New app"**
3. Choose this repository from the list
4. Set the main file as `app.py`
5. Click **"Deploy"**
6. Wait a few minutes — you'll get a link you can share with anyone!

---

## How It Works

- Type or paste any text into the app
- Click **Detect Language**
- The app will tell you which language it is

---

## Languages Supported

- English
- Kiswahili
- Sheng
- Kikuyu
- Luo
- Kalenjin
- Swahili + English mix
- Sheng + English mix

---

## Results

The app correctly identifies the language **94% of the time**.
